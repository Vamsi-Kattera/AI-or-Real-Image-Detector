# AI vs REAL Image Identifier

This project classifies images as **AI-generated** or **Real** using **TensorFlow** and **MobileNetV2**. A Streamlit app allows users to test images interactively.

---

## Features
- Binary classification: AI vs Real
- Transfer learning using MobileNetV2
- Data augmentation for improved generalization
- Streamlit interface for interactive testing
